# Sistema de Formulário Local

# Projeto experimental feito com **React + PHP + Python (Flask)**, simulando o fluxo de cadastro, login, recuperação e envio de mensagens.  
# O sistema funciona em ambiente local (localhost) com banco de dados MySQL no XAMPP.
